# Dation — Web App (монорепо)

Все макеты экосистемы «Dation» в одном репозитории. Каждое приложение — независимый
проект на **React + Vite** со своим `package.json`, ESLint и дев-сервером.

## Структура

```
apps/
├── admin/     # Веб-админка (управление салоном: журнал, клиенты, услуги,
│              #   финансы, уведомления, Telegram Mini App, аналитика)
├── miniapp/   # Telegram Mini App — клиентский контур онлайн-записи
└── pos/       # POS-терминал (журнал список/канбан, оплата, PIN-профили)
```

## Запуск (для каждого приложения)

```sh
cd apps/<admin|miniapp|pos>
npm install
npm run dev      # дев-сервер Vite
npm run build    # прод-сборка
npx eslint src   # проверка на мёртвый код (0 предупреждений)
```

Рекомендуемые порты для одновременного запуска: admin `5180`, miniapp `5181`, pos `5182`.

## Принципы

- Дизайн: фиолетовый бренд `#7C3AED`, белый фон, без эмодзи.
- Правило проекта: при удалении функционала — удалять полностью (UI + код + мок-данные +
  ставшие лишними импорты), добиваясь `0` предупреждений ESLint.
- Это фронтенд-макеты на мок-данных; бэкенд не входит в эти репозитории.

Подробные правила каждого приложения — в его `apps/<name>/CLAUDE.md`.
